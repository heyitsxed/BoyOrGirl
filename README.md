# BoyOrGirl

A simple, Guess Gender app built in Swift

## Features
- Type your name and the app will guess whether it is male or female.
- Displays the number of people who share the same name.
- Shows the percentage indicating whether the name is predominantly male or female.

## Installation
1. Clone the repository:
```bash
git clone https://github.com/heyitsxed/BoyOrGirl
```

2. Open the project in Xcode.
3. Build and run on the simulator or a physical device.

## Usage
1. Enter your name.
2. Click the button (PREDICT).
3. Wait to generate the result.
4. The result card will be displayed on screen.

## Purpose
This app is built as a **personal / pet project** to practice app development.

## Disclaimer
This app provides estimates based on name data and may not be 100% accurate.
